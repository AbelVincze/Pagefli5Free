PDF to Pageflip5 converter + API Read Me First document


FIRST STEP:
Upload the content of the "pageflip5_converter_package" folder to your private webserver
(the converter can't be made available for public usage)

BEFORE START USING THE CONVERTER:
Edit file lib/settings_sample.php, and save it in the same directory as settings.php

THEN CHECK:
load test.php in your browser


USE THE CONVERTER:
load index.html in your browser

TEST CONVERTER API:
edit/load converttest.html and check script feedback in the browser's console
